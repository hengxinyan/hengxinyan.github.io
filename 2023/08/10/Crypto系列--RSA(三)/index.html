[RSA3]P1

扩展欧几里得

裴蜀定理：对于整数域中的不定方程$ax + by = m$，其有解的充要条件为$gcd(a,b) \ | \ m$

$ax_{1}+by_{1}=gcd(a,b)$

$b*x_{2}+(a % b)*y_{2}=gcd(b,(a%b))$

我们知道$a%b=a-(a/b)*b$

$\Rightarrow gcd(a,b) = b*x_{2}+(a-(a/b)*b)*y_{2}$

$\Rightarrow gcd(a,b) = ay_{2}+b(x_{2}-(a/b)*y_{2})$

对应相等一下

$x_{1} = y_{2}$

$y_{1}=x_{2}-(a/b)*y_{2}$

def extgcd(a, b):
    if b == 0:
        return a, 1, 0
    
    g, x, y = extgcd(b, a%b)
    return g, y, x - (a//b)*y
